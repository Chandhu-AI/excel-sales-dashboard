# Excel Sales Dashboard

## Project Overview
Interactive Excel dashboard developed to analyze sales performance across countries, products, and time periods. Designed to provide clear visual insights for business decision-making.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers (Country, Year, Month)

## Key Analysis
- Monthly sales trends  
- Sales contribution by country  
- Product-wise revenue distribution  
- KPI-driven insights using interactive filters

## Business Value
This dashboard enables stakeholders to quickly identify high-performing markets, seasonal trends, and top-selling products, supporting faster and data-driven business decisions.
